### data里面的dsj.json   gsdt.json   gsgg.json

 2. gsdt.json 用于公司动态列表页面的展示
 3. gsgg.json 用于公司公告列表页面的展示
 
 \* 列表页面json数据格式
 
    {
        "title":"这里是标题",
        "date":"这里填写日期",
        "articleUrl":"这里是文章的路径",
        "previewImg":"这里是预览图片的路径",
        "content":"这里是文章的大致内容,或一部分文字内容"
    }
